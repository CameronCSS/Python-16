<a id="about"></a>
# About The Project

This repo tracks my progress through the **Python in 16 Days** Udemy course. After spending time working with **Java** and **JavaScript**, I'm using this course to sharpen my Python fundamentals and get back up to speed with the language.

Each day has its own file covering the concepts from that day's lessons.

## **Tech Stack:** Python

<a id="goals"></a>
### Goals

- Rebuild a strong understanding of Python core fundamentals
- Practice writing clean, readable Python code
- Cover topics I may have gotten rusty on after working in other languages
- Complete all 16 days of the course

<p align="right">(<a href="#readme-top">back to top</a>)</p>


<a id="topics"></a>
# Topics Covered

Topics grow as the course progresses. Expected coverage includes:

- Variables, data types, and type casting
- Conditionals and control flow
- Loops (`for`, `while`)
- Functions and scope
- Lists, tuples, dictionaries, and sets
- String manipulation
- File I/O
- Error handling and exceptions
- Object-oriented programming (OOP)
- Modules and imports
